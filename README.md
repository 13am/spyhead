# spyhead


DESCRIPTION

Spyhead shows the column names and indexes of the input file. E.g.:

> spyhead my_inputfile.txt

> 1 : column_1_name

> 2 : column_2_name

> 3 : column_3_name


REQUIREMENTS

Python 2.7 or newer.

INSTALLATION

Copy the file spyhead.py to a folder. To execute, type:
python path/to/spyhead.py --help
